# Notes Rest

## Work in progress. Do not use.

Rest Documentor generator. Cli utility to generate rest documentation in formats like
blueprint, json, html.

## Install

Via Composer

``` json
{
    "require": {
        "league/notes-rest": "~1.0"
    }
}
```


## Usage
``` json
@Rest({
    "title": "",
    "section": "",
    "request" : {},
    "response": {}
})
```

## Testing

``` bash
$ phpunit
```
